# drift_editor（縦書きPWA対応）
流氷エディタ

このプロジェクトは、青空文庫形式のテキストを縦書きで表示する最小構成のウェブアプリです。

## 機能
- writing-mode: vertical-rl による縦書きCSS
- `<ruby>`タグによるルビ表示
- `<em>`タグによる傍点（text-emphasis）
- PWA対応：ホーム画面追加・オフライン表示（簡易）
- Google Driveへのログイン/ファイル読み込み/変更の取り込み
- 青空文庫形式パーサ（ルビ・傍点）
